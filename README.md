1. すぐ利用できる　やめてもすぐ再開できる　自由にできる

### Discription

Web文字盤で使用している技術
Web文字盤は、Webサイト（ホームページ）を作る際に利用するhtmlとcssに加え、JavaScriptを使って作られています。
このほか合成音声やオフラインWebアプリなど先人が開発し無償で公開している技術を利用しています。このようにすでによく知られている、先進的ではない技術を用いて作られていますので、とくに費用はかかっていません。
これは手紙を書くとき使う文字や、散歩するときに歩く歩道が無料なのと同じことです。ただ使用するにはいくらか知識が必用なだけです。

また、無料のgithubpagesを公開に使用していますのでこれも費用はかかっていません。
このようにインターネットで誰でも手に入る知識や材料を使い、
必用な人が誰でも自由に利用できるツールを無償で提供できるようになります。
